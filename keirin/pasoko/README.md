# 競輪予想AI「パソ子」

## 概要

本システムは、過去（2018年〜2025年）の出走表データおよびレース結果を取得し、各レース内における選手の相対的な能力や「ライン」の強度を独自に特徴量化して学習するAIモデルです。

**最大の特徴**: 人間の集合知である「オッズ」や「人気順」を特徴量から完全に排除することで、オッズが確定する前の未来のレース（今日以降の開催）の事前予測を可能としています。

## インストール

```bash
cd C:\keirin_ai\pasoko
pip install -r requirements.txt
```

## 使用方法

### デモモード（ダミーデータで動作確認）
```bash
python src/main.py --mode demo
```

### 全処理実行（データ収集→学習→予測）
```bash
python src/main.py --mode all
```

### 個別処理

```bash
# 1. データスクレイピング（2018年〜2025年）
python src/main.py --mode scrape

# 2. 特徴量エンジニアリング
python src/main.py --mode feature

# 3. モデル学習
python src/main.py --mode train

# 4. 予測実行
python src/main.py --mode predict              # 本日
python src/main.py --mode predict --date 20240315  # 指定日
```

## 機能詳細

### 1. データ取得機能
- 2018年〜2025年の全レース情報を取得
- 出走表（race_card）、レース情報（race_info）、払戻情報（race_return）の3種類
- サーバー負荷軽減のため1秒間隔でアクセス
- 開催中止やレース中止はスキップ処理

### 2. 特徴量エンジニアリング
- **バンク周長**: 333/400/500バンクの分類
- **相対的ステータス**: レース内での差分（最大値との差、上位2名の差など）
- **ライン構成**: 車立、ライン数、ライン構成、番手
- **ライン強度**: 独自指標（a1, b2, c3形式）
- **フラグ4**: ライン先頭で競走得点・バック回数が両方1位の強力な先行指標

### 3. AIモデル
- **学習手法**: 二値分類（3着以内: 1, それ以外: 0）
- **モデル分類**:
  - 7車立て用 / 9車立て用 / イレギュラー
  - ガールズケイリン用 / チャレンジレース用
  - G3特化型 / G1特化型
- **回収率対策**: あえて精度を落とす調整（キャリブレーションカーブのズレ）

### 4. 独自指標
- **A率**: ランクAの選手の3着以内確率（0.9超で1着率約75%）
- **KS値**: B率 - C率（1.0超でAB突出）
- **CT値**: ABC決着確率（0.8超で本命サイド）

### 5. ゾーン分類
| ゾーン | 条件 | 推奨戦略 |
|--------|------|----------|
| ガチゾーン | A率≧0.9, CT≧0.8 | 3連複1点（ABC） |
| ブルーゾーン | CT≧0.75 | 3連複2点（ABC, ABD） |
| トワイライト | CT≧0.50 | 3連複3点 |
| レッドゾーン | CT<0.50 | A軸全流し or efg絡み |

### 6. 宝探しロジック
- **条件A（超鉄板）**: A率≧0.9, CT≧0.8, KS≧1.0 → 回収率約300%
- **条件B（紐荒れ）**: A率≧0.9, CT<0.5 → A軸からDEF流し

### 7. イカサマサイコロ
- AIの確率を重みとした重み付きランダム抽選
- 確率を2乗して重みに使用（強い選手の出現率UP）
- CT値に応じて試行回数を可変

## 除外データ
- 落車・失格が発生したレース
- 新人戦

## 注意事項

1. **競りの限界**: AIは「競り」を予測パラメーターにしていないため、明確な競りやイン粘りが予想される場合は人間の判断を加えてください。

2. **細の型（特定パターン）**: G1/G2の初日において細切れ戦で「4, 6, 8」が複数いる場合、システム予測の補完として独自の車券術が有効な場合があります。
